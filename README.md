### 코드 블럭(Code block)

1. 코드박스

```python
class = '부울경 2반'

```

2. 인라인코드
   * "`"(백킷) 안에 코드/문서 내용을 작성해준다.
     * `printf('부울경 2반')` -> `print('부울경 2반')`

### 링크(link)

### 택스트 강조(text emphasis)

* 굵게
  * `*`를 텍스트 앞뒤에 2개씩 씌어준다.
  * **부울경**
* 기울임
  * *부울경*
  * `*`를 텍스트 앞뒤에 씌어준다.
* 취소선
  * ~~구미홍개라면~~
  * `~`를 텍스트 앞뒤에 2개씩 씌어준다.

---

### 수평선(horizontal line)

* `-` 을 3개 입력한다.
* `<hr>`을 입력한다.
